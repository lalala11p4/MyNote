###### (3)cd指令

​	当前工作目录/root,希望进入到/home下

​	绝对路径：	/home/     即从根目录开始定位

​	相对路径：	../home	从当前工作目录开始定位到需要的目录

​	cd ~  或者 cd	回到自己的家目录

​	cd..	回到当前目录的上一级目录

![1563610042448](E:\Typora笔记\Pic\1563610042448.png)

​	例1：绝对路径到root

​		cd /root

​	例2：使用相对路径到root目录

​		我们需要知道当用户在哪个目录下	假设我们现在在 /usr/lib 文件夹下

​		cd ../../root

![1563614106730](E:\Typora笔记\Pic\1563614106730.png)

​	例3：表示回到当前目录的上一级目录

​		cd ..

![1563614291697](E:\Typora笔记\Pic\1563614291697.png)

​	例4：回到家目录

​		cd

​		cd ~

![1563614789498](E:\Typora笔记\Pic\1563614789498.png)

###### (7)cp(重要)复制指令

​	-r      ：递归复制整个文件夹

​	例：将   /home/aaa.txt  拷贝到  /home/bbb目录下(拷贝单个文件)

​		cp  aaa.txt bbb

![1563637509504](E:\Typora笔记\Pic\1563637509504.png)

​	例：递归复制整个文件夹

​			将home/test  整个目录  拷贝到  zhngwuji目录下

![1563714275517](E:\Typora笔记\Pic\1563714275517.png)

###### (11)less指令（用于分屏查看文件）

​	less指令用于分屏查看文件，在查看文件时根据显示需要加载内容，对现实大型文件有较高的效率

​	less   文件全名

​	快捷键：

​		space	： 向下翻一页

​		Enter	 ： 向下走一行

​		q			： 立即离开

​		ctrl	+	f	：向下一屏

​		ctrl	+	b	：返回上一屏

​		=	：行号

​		f	 ：输出文件名和当前行的行号
